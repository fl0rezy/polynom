# Лабораторная работа "Polynomial"
Выполнили: Вознесенский Дмитрий, Грушецкий Максим
# Цель работы
Создание базы данных для хранения полиномов с поддержкой операций добавления, удаления и поиска, а также исследование эффективности различных структур данных для решения данной задачи.

# Используемые инструменты
Git;

Google Test;

Microsoft Visual Studio;

CMake

# Задачи
1. Реализовать класс полиномов с арифметическими операциями.

2. Реализовать транслятор для полиномов.

3. Реализовать базы данных: упорядоченные и неупорядоченные массивы, хеш-таблицы (метод цепочек и открытая адресация), поисковые деревья (AVL-дерево и красно-черное дерево).

4. Реализовать систему тестирования с использованием Google Test.

5. Разработать интерфейс.

# План
1. Разработка класса Полином
